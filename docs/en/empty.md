### Attributes

| 参数        | 说明              | 类型   | 可选值 | 默认值            |
| ----------- | ----------------- | ------ | ------ | ----------------- |
| size        | 自定义尺寸        | string | -      | medium/small/mini |
| class       | 自定义 class 样式 | string | -      | -                 |
| image       | 图片地址          | string | —      | —                 |
| imageSize   | 图片大小（宽度）  | number | —      | —                 |
| description | 文本描述          | string | —      | —                 |

### Section

| 名称        | 描述           |
| ----------- | -------------- |
| default     | 自定义底部内容 |
| image       | 自定义图片     |
| description | 自定义描述文字 |
