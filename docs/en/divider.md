### Attributes

| 参数             | 说明                 | 类型   | 可选值                | 默认值     |
| ---------------- | -------------------- | ------ | --------------------- | ---------- |
| class            | 自定义 class 样式    | string | -                     | -          |
| direction        | 设置分割线方向       | string | horizontal / vertical | horizontal |
| content-position | 设置分割线文案的位置 | string | left / right / center | center     |
| background       | 背景色               | string | -                     | -          |
